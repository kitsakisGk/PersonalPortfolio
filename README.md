# Georgios Kitsakis — Personal Portfolio

Personal portfolio website built with Next.js 15, Tailwind CSS, and Framer Motion. Designed as a client-acquisition tool showcasing Data Engineering, ML, and Web Development work.

## Tech Stack

- **Framework:** Next.js 15 (App Router, TypeScript)
- **Styling:** Tailwind CSS + inline CSS with custom design tokens
- **Animations:** Framer Motion (scroll-triggered, staggered)
- **Fonts:** Geist Sans + Geist Mono
- **Hosting:** Vercel (CI/CD — every push to `main` auto-deploys)

## Sections

- **Hero** — Name, roles, stats bar, CTAs
- **About** — Bio, experience timeline, education
- **Skills** — Tech stack grouped by domain, human languages with CEFR levels
- **Projects** — Featured and secondary projects with tags
- **Certifications** — 13 credentials grouped by category, linked to Google Drive
- **Contact** — Email CTA, LinkedIn, GitHub

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Connected to Vercel — every push to `main` triggers an automatic production deploy.
