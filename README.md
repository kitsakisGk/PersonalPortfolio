## Overview

A cutting-edge personal portfolio built as a client-acquisition tool. Designed with a near-black background and deep forest green accent palette, featuring scroll-triggered animations, glassmorphism cards, and a fully responsive layout. Showcases Data Engineering, ML Engineering, and Web Development work.

## Features

- **Scroll Animations** — Staggered fade-up reveals powered by Framer Motion
- **Responsive Design** — Mobile-first, works seamlessly on all screen sizes
- **Dark Theme** — Near-black background with deep forest green (`#255736`) accents
- **6 Sections** — Hero, About, Skills, Projects, Certifications, Contact
- **SEO Ready** — OpenGraph, Twitter card, sitemap, robots.txt
- **CI/CD** — Every push to `main` auto-deploys to Vercel

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | Next.js 15 (App Router, TypeScript) |
| Styling | Tailwind CSS + CSS custom properties |
| Animations | Framer Motion (whileInView, stagger) |
| Fonts | Geist Sans + Geist Mono |
| Deployment | Vercel (CI/CD) |

## Sections

- **Hero** — Name, roles, stats bar (2+ years, 6+ projects, 13 certs, MSc), CTAs
- **About** — Bio, experience timeline (Intrum, Accenture, DeltaFoods), education
- **Skills** — 7 skill groups + human languages with CEFR levels
- **Projects** — 6 projects (3 featured), tags, GitHub links
- **Certifications** — 13 credentials grouped by category, linked to Google Drive
- **Contact** — Email CTA, LinkedIn, GitHub, footer

## Getting Started

```bash
git clone https://github.com/kitsakisGk/PersonalPortfolio.git
cd PersonalPortfolio
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Connected to Vercel — every push to `main` triggers an automatic production deploy. No manual steps needed.
